    Python Custom Formatting

    Python Custom Formatting

    Python f-strings use a formatting mini-language, the same as the older .format() function. After the colon comes short specifications for how to format the value:

    >>> word = "Hello"
>>> f"{word:/^20}"
'///////Hello////////'
>>> amt = 12345678
>> f"{amt:20,}"
'          12,345,678'
